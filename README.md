## Description of Scripts

### Script 1 – System Identity Report
This script displays basic system information such as kernel version, logged-in user, uptime, and current date. It acts like a system introduction.

### Script 2 – Package Checker
This script checks whether Git is installed on the system. If installed, it shows the version and provides a short explanation of the tool.

### Script 3 – Directory Auditor
This script examines important system directories and shows their size along with permissions and ownership details.

### Script 4 – Log Analyzer
This script reads a log file and counts how many times a specific keyword (like "error") appears. It also displays recent matching lines.

### Script 5 – Manifest Generator
This script takes user input and generates a personalized open-source statement, which is saved in a text file.

## Steps to Run the Scripts

1. Open terminal  
2. Give permission to run scripts:
   chmod +x script1.sh  

3. Run the script:
   ./script1.sh  

(Same process can be followed for all scripts)
